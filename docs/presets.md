# 预设场景参数对照表

本系统为四大抗捕食机制分别设计了 A/B 对比预设。用户点击预设按钮后，系统自动调整所有相关参数并重置仿真。

---

## 1. 多眼效应 (Many-Eyes)

| 参数 | A: 个体模式 | B: 群体模式 | 说明 |
|------|------------|------------|------|
| num_boids | 200 | 200 | 保持一致 |
| num_predators | 1 | 1 | 保持一致 |
| visual_range | **30** | **100** | 核心变量：视觉范围 |
| matching_factor | **0.01** | **0.08** | 核心变量：对齐强度 |
| centering_factor | 0.005 | 0.005 | 保持一致 |
| confusion_threshold | 999 | 999 | 禁用混淆 |
| prefer_edge | false | false | - |

**预期结论**: B 方案中警报传播速度显著快于 A，体现在平均反应时间指标的差异。

---

## 2. 稀释效应 (Dilution)

| 参数 | A: 小群 | B: 大群 | 说明 |
|------|--------|--------|------|
| num_boids | **30** | **300** | 核心变量：群体大小 |
| num_predators | 1 | 1 | 保持一致 |
| visual_range | 75 | 75 | 保持一致 |
| confusion_threshold | 999 | 999 | 禁用混淆（隔离稀释效应） |

**预期结论**: 虽然 B 的总捕获数可能更高，但个体平均被捕率（捕获数/N）显著低于 A。

---

## 3. 混淆效应 (Confusion)

| 参数 | A: 无混淆 | B: 有混淆 | 说明 |
|------|----------|----------|------|
| num_boids | 200 | 200 | 保持一致 |
| num_predators | 1 | 1 | 保持一致 |
| confusion_threshold | **999** | **3** | 核心变量：混淆阈值 |

**预期结论**: B 方案中捕食者攻击成功率显著低于 A，目标切换频率显著高于 A。

---

## 4. 自私兽群 (Selfish Herd)

| 参数 | A: 松散群体 | B: 紧密群体 | 说明 |
|------|-----------|-----------|------|
| num_boids | 200 | 200 | 保持一致 |
| centering_factor | **0.001** | **0.01** | 核心变量：聚合强度 |
| avoidance_factor | **0.15** | **0.03** | 核心变量：分离强度 |
| prefer_edge | true | true | 两组都启用边缘攻击 |

**预期结论**: B 方案中群体更紧密（紧密度指标更低），边缘个体被捕比例更高——验证了中心位置更安全的论点。

---

## 使用建议

1. 先点击某机制的 **A 预设**，观察 30~60 秒，记录统计指标
2. 再切换到 **B 预设**，同样观察相同时间
3. 对比统计仪表盘中的关键指标差异
4. 可以手动微调参数，探索临界值和边界条件
