# 四大抗捕食机制原理说明

## 概述

鸟群（Boids）模型是 Craig Reynolds 于 1986 年提出的经典群体行为仿真算法。本系统在经典三规则（分离、对齐、聚合）基础上，引入捕食者机制，用于验证和演示四种经典的抗捕食策略。

---

## 1. 多眼效应 (Many-Eyes Effect)

### 原理

群体中总有个体在警戒。每个个体可以减少自身警戒时间（多花时间觅食），但群体整体对捕食者的侦测反而更快、更远。

### 与鸟群模型的关系

- 每只鸟只需监控周围少数邻居的行为变化
- 当一只鸟检测到捕食者并改变飞行方向时，通过**对齐规则**将惊恐状态传播给邻居
- 形成链式反应，实现快速群体响应

### 仿真中的体现

- **警戒传播**: 当一只鸟的 `alert_level` 升高时，通过对齐机制传播给视觉范围内的邻居
- **关键参数**: `visual_range`（视觉范围）越大，信息传播越快；`matching_factor`（对齐权重）越高，响应越迅速
- **观测指标**: 平均反应时间——从第一只鸟发现捕食者到全群进入警戒状态的帧数

## 2. 稀释效应 (Dilution Effect)

### 原理

捕食者一次只能捕获一只猎物，群体越大，每个个体被选中的概率越低（约为 1/N）。

### 理论基础

在大群体中，个体被捕概率随群体大小 N 的增加而线性下降。这是群聚行为最直接的数学优势。

### 仿真中的体现

- 捕食者每次只追击并尝试捕获一个目标
- **关键参数**: `num_boids`（鸟群数量）直接影响个体被捕概率
- **观测指标**: 单位时间捕获次数 / 鸟群数量 = 个体平均被捕率

---

## 3. 混淆效应 (Confusion Effect)

### 原理

大群体高速运动时，捕食者因感知和认知瓶颈难以锁定单一目标，攻击成功率随群体大小/密度增加而下降。

### 实验证据

对三维模拟椋鸟群的实验表明，捕食者的命中率随猎物群数量增加而显著降低。这源于视觉处理的瓶颈——当大量相似目标快速移动时，选择性注意力失效。

### 仿真中的体现

- 当捕食者感知范围内的猎物数量超过 `confusion_threshold` 时:
  - 锁定成功率按 `1/√N` 衰减
  - 捕食者有概率随机切换目标（而非持续追击同一个体）
- **关键参数**: `confusion_threshold`（混淆阈值）
- **观测指标**: 攻击成功率（成功捕获 / 攻击尝试次数）、目标切换频率

## 4. 自私兽群 (Selfish Herd)

### 原理

个体通过挤向群体中心来降低自身处于边缘的暴露风险。边缘个体的"危险域"（Voronoi 多边形面积）最大，被捕概率最高。

### 理论背景

Hamilton (1971) 用几何模型证明：中心个体的被捕概率最低。后续研究发现，简单的"向最近邻靠拢"规则可以将被捕风险降低最多 67%。

### 仿真中的体现

- 增强 `centering_factor`（聚合权重）使鸟群更紧密
- 捕食者开启 `prefer_edge_target` 模式时，优先攻击距群体质心最远的个体
- **关键参数**: `centering_factor` 和 `avoidance_factor` 的比值决定群体紧密度
- **观测指标**: 边缘个体被捕比例 vs 中心个体被捕比例、群体紧密度
